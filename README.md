# Gorilla Tag Mod

A custom modification for Gorilla Tag VR game.

## Features
- Custom gameplay mechanics
- Enhanced visuals
- Quality-of-life improvements

## Installation
1. Download the latest release
2. Extract to your Gorilla Tag mods folder
3. Launch the game

## Building
Requirements:
- Unity 2019 LTS or later
- Gorilla Tag SDK
